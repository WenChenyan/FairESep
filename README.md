# FairESep: Causal Dual-View Edge Separation for Fair Recommendation

## Repository Status

At the current submission stage, this repository provides the datasets and related experimental resources used in our paper.
The full source code of FairESep will be released after the paper is accepted.

## Datasets

The datasets used in our experiments are provided under the `data/` directory.

```text
Dataset/
├── ML-1M/
├── ML-100K/
├── LastFM-360K/
└── IJCAI-2015/
```

Each dataset directory contains the processed interaction data and sensitive-attribute information used for fair recommendation evaluation.

## Baseline Reference

Some baseline settings and experimental implementations refer to the public FairDgcl repository:

[https://github.com/cwei01/FairDgcl](https://github.com/cwei01/FairDgcl)

We thank the authors for releasing their implementation.

## Code Release

The implementation of FairESep, including model training, evaluation scripts, and configuration files, will be made publicly available after the paper is accepted.
